# 個人記帳app - 用戶交互設計

## 核心交互功能

### 1. 快速記帳功能
- **新增收支記錄**：用戶點擊「新增記錄」按鈕，彈出表單包含：
  - 金額輸入框（必填，數字鍵盤）
  - 收支類型選擇器（收入/支出）
  - 分類下拉選單（餐飲、交通、購物、娛樂、薪資等）
  - 日期選擇器（預設今天）
  - 備註輸入框（選填）
- **表單驗證**：金額必填且為正數，其他欄位提供合理預設值
- **快捷操作**：支援Enter鍵快速提交，ESC鍵取消

### 2. 收支記錄管理
- **記錄列表顯示**：
  - 以卡片形式展示每筆記錄
  - 每筆記錄顯示：日期、分類圖標、金額、備註
  - 收入用綠色標示，支出用紅色標示
  - 支援按日期排序（最新在前）
- **編輯功能**：點擊記錄卡片可進入編輯模式
- **刪除功能**：長按或滑動顯示刪除選項，需二次確認
- **搜索過濾**：
  - 按日期範圍過濾
  - 按分類過濾
  - 按關鍵字搜索備註

### 3. 數據分析視覺化
- **月度統計圖表**：
  - 使用ECharts展示每月收支趨勢折線圖
  - 可切換時間範圍（3個月、6個月、1年）
  - 滑鼠懸停顯示詳細數據
- **分類圓餅圖**：
  - 展示各分類支出佔比
  - 點擊圖表切片可查看該分類詳細記錄
- **統計卡片**：
  - 當月收入、支出、結餘
  - 與上月的增減百分比
  - 動畫數字滾動效果

### 4. 預算管理系統
- **設定預算**：
  - 為每個支出分類設定月度預算
  - 預算金額輸入和修改
  - 預算狀態視覺化（進度條）
- **預算追蹤**：
  - 實時顯示各分類已使用金額
  - 超支警告（紅色標示）
  - 預算使用率百分比顯示
- **預算提醒**：
  - 當某分類支出達到預算80%時提醒
  - 自定義提醒閾值設定

### 5. 數據導出與備份
- **CSV導出**：一鍵導出所有記錄為CSV格式
- **數據清理**：批量刪除指定日期範圍的記錄
- **本地存儲**：使用localStorage保存數據，確保刷新頁面不丟失

## 用戶操作流程

### 主要使用場景
1. **日常記帳**：打開app → 點擊新增 → 填寫金額和分類 → 保存
2. **查看統計**：切換到分析頁面 → 查看圖表和統計數據
3. **預算管理**：切換到預算頁面 → 設定各分類預算 → 查看使用情況
4. **記錄管理**：在主頁面瀏覽記錄 → 編輯或刪除不需要的記錄

### 交互細節
- **響應式設計**：適配手機和桌面設備
- **觸控友好**：按鈕大小適合手指點擊
- **鍵盤快捷鍵**：桌面版支援快捷鍵操作
- **無障礙設計**：支援螢幕閱讀器和鍵盤導航
- **載入動畫**：數據載入時顯示優雅的loading效果
- **錯誤處理**：網路錯誤或數據錯誤時提供友好提示

## 技術實現要點
- 使用Anime.js實現平滑的頁面切換和元素動畫
- 使用ECharts創建互動式圖表
- 使用localStorage實現本地數據持久化
- 使用響應式設計確保多設備兼容性
